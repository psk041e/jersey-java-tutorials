#Jersey API를 사용한 REST 웹 서비스 작성

#Softwares and Tools
	- jdk1.8.0_202 64bit
	- apache maven embeded
	- tomcat 8.5.57 64bit
	- eclipse IDE for Java EE 2018-09 64bit
	
#step
1. Dynamic Java Project 생성
	- maven 프로젝트로 변환
2. dependency 추가: jersey-client 1.19.4 
3. 클래스 생성
4. 빌드해서 war 파일을 만들고 애플리케이션 서버에 배포
